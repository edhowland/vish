### Ruby installation and setup

If you do not yethave  Ruby on your system, follow these steps

```
# For Debian-flavored Linux'es
$ sudo apt-get update
$ sudo apt-get install ruby     build-essential   libjson-glib-dev   ruby-dev 
```
